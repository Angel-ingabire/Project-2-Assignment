# Question 2: Airport Security Max-Heap Priority Queue

---

## Problem Description

In a smart airport security control system, passengers are screened based on a **risk assessment score** generated by automated checks. Passengers with **higher risk scores** must be processed **first** for additional security screening.

As passengers arrive at the checkpoint, their details are recorded and **dynamically prioritized** using a **Max-Heap**.

Each passenger has:
- **Passenger name** (e.g., Alice, Brian, Chen, Fatima, Noah, Sofia)
- **Risk score** (integer value)

---

## Tasks

### Task 1: Build a Max-Heap

- Build a Max-Heap from the list of passengers.
- The passenger with the **highest risk score** must always be at the **root**.
- Use bottom-up `heapify-down` (array-based binary heap).

### Task 2: Extract priority queue (screening order)

- Using the Max-Heap, extract a **priority queue** that represents the order in which passengers will be screened.
- Order must be **highest risk first** (repeated `extract-max` operations).
- Extraction is performed on a **copy** of the heap so the original heap remains intact.

### Task 3: Insert passenger with risk score 98

- A new passenger arrives with risk score **98**.
- Insert into the heap and re-adjust with **heapify-up**.

### Task 4: Remove passenger with risk score 98

- The passenger with risk score **98** is cleared.
- Remove from the heap and re-adjust with **heapify-down** (or heapify-up if needed).

## Implementation Details

### Data structures

```c
typedef struct {
    char name[NAME_LEN];

```

### Core operations

| Function | Purpose |
|----------|---------|
| `build_max_heap()` | Bottom-up heapify to create max-heap |
| `extract_max()` | Remove and return highest-risk passenger |
| `extract_priority_queue()` | Extract all passengers in screening order (on a copy) |
| `insert_passenger()` | Append passenger and heapify-up |
| `remove_passenger_by_risk_score()` | Delete by risk score and re-heapify |
| `verify_max_heap()` | Validates max-heap property |

### Array-based heap indexing

| Operation | Formula |
|-----------|---------|
| Parent of index `i` | `(i - 1) / 2` |
| Left child of `i` | `2 * i + 1` |
| Right child of `i` | `2 * i + 2` |
| Last non-leaf index | `(size / 2) - 1` |
